🧮 Calculatrice en Python
Une calculatrice en ligne de commande réalisée en Python. Ce projet permet d'effectuer les quatre opérations arithmétiques de base grâce à un menu interactif.

🚀 Fonctionnalités
➕ Addition
➖ Soustraction
✖️ Multiplication
➗ Division
🚪 Quitter le programme
📂 Structure du projet
calculatrice_simple/
│
├── calcul.py
└── README.md
▶️ Exécution
Prérequis
Python 3.x installé sur votre ordinateur.
Lancer le programme
python calculator.py
💻 Exemple d'utilisation
entrez le premier nombre : 10
entrez le second nombre : 5

==========MENU=========
1. Addition
2. Soustraction
3. Multiplication
4. Division
5. Quitter

entrez le choix : 1

Le résultat est : 15
